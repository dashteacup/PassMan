Manual Test Plan

Make a new file via the tool bar

  1. Open the app. You should see:
  2. Click the New file button on the toolbar
  3. A new empty text area should open up.
  4. Enter the text "Hello world!"
  5. Click the Save As button on the toolbar
  6. The save as dialog box should load. Enter the file name "newname.pman" and hit the save button. (This test assumes there is no file already named "newname.pman").
  7. The enter password dialog should appear. Enter "monkey" in the field and click Ok
  8. Now close the app.
  9. Restart the app and click the open file button on the toolbar
  10. The open file dialog box should appear. Select the "newname.pman" file and click open.
  11. The enter password dialog will pop up again. Enter the password "monkey" one more time.
  12. The file we created earlier should now show up in the text area again.
